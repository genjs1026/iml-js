iml-js - sample project
===

HOW-TO
---

Download the sample project :
```
git clone https://github.com/lchaboud/iml-js.git
cd iml-js
cd sample
```

Initialize and run :
```
npm install
node main.js
```

You get this result :
```
{ 'My recipes': [ { 'Best Chocolate Chip Cookies': [Object] } ] }
```
